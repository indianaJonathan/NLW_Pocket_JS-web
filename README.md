## pt-BR

Esse projeto depende do [backend](https://github.com/indianaJonathan/NLW_Pocket_JS-server) para funcionar corretamente

### Configurações de desenvolvimento
Instale as dependências
```shell
npm install
```

Inicie a aplicação
```shell
npm run dev
```
